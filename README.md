# ICT Talent Pipeline - 

A professional dashboard for managing the national ICT talent lifecycle in Kenya.

## Features
- **Executive View**: National deployment heatmap with county-level drill-down.
- **Operational Kanban**: Drag-and-drop talent verification workflow.
- **Skills Analytics**: Automated gap analysis for PSC and Private Sector placement.
- **Secure Data**: PostgreSQL integration for persistent talent tracking.

## Tech Stack
- React + Vite
- Tailwind CSS
- D3.js / React-Simple-Maps
- Context API for Global State
- Supabase
